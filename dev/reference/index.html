Pastebin.delpaste! — Method
delpaste!(client, key)

Deletes paste key from the server. Requires a userKey.

source
Pastebin.getraw — Method
getraw(key)

Fetch a raw paste by key. Returns a response = Pastebin.Response(...) object. The raw paste string can be accessed as Pastebin.content(response).

source
Pastebin.paste! — Method
paste!(client, title, text; expire = "N", access = 0)

Submits text to pastebin. expire date is one of: "N" (never, default), "10M" (10 minutes), "1H" (1 hour), "1D" (1 day), "1W" (1 week), "2W" (2 weeks), "1M" (1 month), "6M" (6 months), "1Y" (1 year). access is an integer: 0 (public), 1 (unlisted, default) or 2 (private, requires a userKey). format determines syntax highlight (plain text by default), see https://pastebin.com/doc_api.
